# 《弥米尔协议》- The Mimir Protocol

一个基于文本的悬疑解谜游戏，采用 SCP 基金会风格的叙事方式。

## 🎮 如何开始

1. 打开 `index.html` 文件（可直接用浏览器打开）
2. 输入雇员 ID：`98-7344` 开始游戏
3. 使用 SEARCH 功能查找文档，逐步揭开真相

## 📚 游戏玩法

### 基本操作
- **搜索文档**：在搜索框输入文档编号（如 `SYS-001`）或关键词（如 `指南`、`thorne`）
- **查看文档**：点击搜索结果即可阅读完整内容
- **返回搜索**：按 `ESC` 键或点击"返回搜索"按钮
- **关键词跳转**：点击文档中高亮的关键词可自动搜索相关内容

### 特殊命令
- `LOGIN [雇员ID]` - 切换用户账号
- `LOGOUT` - 退出系统
- `HELP` - 显示帮助信息

## 🔐 安全等级系统 (SCL)

游戏分为 5 个安全等级，等级越高可访问的文档越多：
- **SCL-1**: 基础档案，建立世界观
- **SCL-2**: 限制档案，开始接触异常
- **SCL-3**: 机密档案，核心冲突浮现
- **SCL-4**: 秘密档案，揭示灾难真相
- **SCL-5**: 绝密档案，最终结局

## 🕵️ 游戏提示

1. **仔细阅读文档**：有些文档中隐藏着提升权限的线索
2. **注意特殊标记**：
   - `[数据删除]` - 需要更高权限才能查看
   - `[数据损坏]` - 信息已损坏
   - 高亮关键词 - 可点击搜索相关内容
3. **追踪人物关系**：
   - Dr. Aris Thorne（激进研究员）
   - Dr. Lena Vance（伦理委员会）
   - Director Kaelen（站点主管）
4. **关注 B4 层**：大部分关键事件发生在此

## 🎯 推荐探索路线

1. 从 `SYS-001` 开始了解基本设定
2. 搜索 `vance` 或 `thorne` 追踪主线剧情
3. 阅读 `ITEM-102` 获取第一个隐藏账号
4. 使用新账号解锁更高等级文档
5. 搜索 `共鸣` 或 `resonance` 了解核心事件
6. 最终抵达 SCL-5 揭开真相

## ⚠️ 理智系统
