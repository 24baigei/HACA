# 《弥米尔协议》- The Mimir Protocol

一个基于文本的悬疑解谜游戏，采用 SCP 基金会风格的叙事方式。

## 🎮 如何开始

1. 打开 `` 文件（可直接用浏览器打开）
2. 输入雇员 ID：`98-7344` 开始游戏
3. 使用 SEARCH 功能查找文档，逐步揭开真相

## 📚 游戏玩法

### 基本操作
- **搜索文档**：在搜索框输入文档编号（如 `SYS-001`）或关键词（如 `指南`、`thorne`）
- **查看文档**：点击搜索结果即可阅读完整内容
- **返回搜索**：按 `ESC` 键或点击"返回搜索"按钮
- **关键词跳转**：点击文档中高亮的关键词可自动搜索相关内容

### 特殊命令
- `LOGIN [雇员ID]` - 切换用户账号
- `LOGOUT` - 退出系统
- `HELP` - 显示帮助信息
